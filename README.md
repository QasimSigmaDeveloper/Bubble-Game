# Bubble-Game
Developed an interactive Bubble Game using HTML, CSS, and JavaScript to enhance DOM manipulation and event handling skills. Implemented dynamic bubble generation and click-based scoring logic to create an engaging user experience.

# Features
Dynamic bubble generation
Real-time score update system
Simple and interactive UI
Built using pure JavaScript (no frameworks)

# Technologies Used
HTML5
CSS3
JavaScript
